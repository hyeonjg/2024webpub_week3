삶은 감자

2021년 11월

레시피

  1. 감자를 씻는다.
  2. 감자가 물에 잠길 만큼 냄비에 물을 넣는다.
  3. 소금을 적당량 뿌린다.
  4. 젓가락으로 찔러서 들어가면 완성!

후기: 냄비가 타서 엄마한테 혼났다.

TIP: 감자가 덜 익어서 물을 더 넣더라도 소금은 더 넣으면 안 된다.

블루베리 두부 스테이크

2021년 7월

레시피

  1. 두부에 칼집을 넣고 기름을 바른다.
  2. 에어프라이어에서 두부가 노릇하게 익을 때까지 굽는다.
  3. 블루베리와 와인, 소금 등을 넣고 끓인다.
  4. 익은 두부스테이크에 스테이크 소스를 얹으면 완성!

후기: 달고 싱거웠다.

고추기름 볶음밥과 양파 튀김

2021년 8월

레시피

  1. 고추기름에 파, 버섯, 양파, 마늘을 넣고 볶는다.
  2. 야채가 어느정도 익으면 밥을 넣고 같이 볶는다.
  3. 달걀 푼 물을 같이 볶는다.
  4. 얇게 썬 양파를 씻어서 한겹한겹 분리해주고 밀가루옷을 얇게 입힌다.
  5. 양파를 후라이팬에서 튀기듯이 굽는다.
  6. 곁들여서 완성!

후기: 맛있게 매웠고 양파 튀김이 바삭해서 좋았다.

TIP: 볶음밥은 갓 지은 밥이 아니라 식은 밥이나 데운 밥으로 만들어야 뭉치지 않고 잘 볶아진다.

토마토 카레와 템페 튀김

2022년 6월

레시피

  1. 양파를 먼저 잘 익은 갈색이 될 때까지 익혀준다.
  2. 깍둑썰기한 토마토와 얇게 썬 양송이를 넣어서 뚜껑을 덮고 익힌다.
  3. 토마토에서 물이 어느정도 나왔으면 카레가루와 스프 스톡을 넣고 다시 뚜껑을 덮고 익힌다.
  4. 템페는 길게 썰어 튀기듯이 굽는다.
  5. 부글부글 끓은 카레와 템페를 예쁘게 접시에 올리면 완성!

후기: 토마토가 적당히 새콤한 맛을 더해줘서 맛있었다.

TIP: 토마토에서 물이 많이 나오기 때문에 카레에 물을 넣을 필요가 없다.

가지 그라탕

2022년 12월

레시피

  1. 가지를 얇게 썰어 살짝 노릇하게 굽는다.
  2. 넣고 싶은 야채들을 잘게 썰어서 기름에 볶는다.
  3. 익은 야채와 토마토소스를 섞어서 한 번 더 볶는다.
  4. 야채가 들어간 토마토소스와 익은 가지를 그라탕 용기에 번갈아서 담는다.
  5. 모짜렐라 치즈를 올리고 전자레인지에서 치즈가 녹을 때까지 돌린다.
  6. 손에 화상을 입지 않게 조심해서 꺼내면 완성!

후기: 가지를 편식하는 나도 맛있게 먹을 수 있었다.

TIP: 가지가 기름을 잘 빨아들여서 기름은 조금만 뿌려도 된다.
